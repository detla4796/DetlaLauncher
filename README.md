# DetlaLauncher

Лаунчер для управления библиотекой игр в стиле старого Steam.

## Описание

DetlaLauncher — это десктопное приложение на WPF для каталогизации игр. Позволяет добавлять игры с описанием, рейтингом, жанром и ссылками на источники. Дизайн стилизован под классический интерфейс Steam с армейским зелёным цветом.

## Функции

- **Добавление игр** — название, жанр, рейтинг (1-10), описание, источники
- **Редактирование игр** — изменение данных существующих игр
- **Удаление игр** — с подтверждением
- **Поиск** — фильтрация по названию
- **Сортировка** — по названию или рейтингу
- **Избранное** — отметка любимых игр и быстрая фильтрация
- **Счётчик** — отображение количества игр ("Показано: X из Y")
- **Сохранение** — данные хранятся в JSON-файле

## Технологии

- C# / WPF (.NET)
- JSON для хранения данных
- Кастомные стили всех элементов (Button, TextBox, ComboBox, CheckBox, ScrollBar)

## Запуск

1. Открыть проект в Visual Studio
2. Запустить (F5)
3. Данные сохраняются автоматически в `games.json`
